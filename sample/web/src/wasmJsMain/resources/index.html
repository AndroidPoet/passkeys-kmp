passkeys-kmp · browser (Wasm) demo

Runs WasmJsPasskeyClient against navigator.credentials. No backend: options are generated client-side and the response is shown but not verified. For testing without a real fingerprint sensor, open DevTools → ⋮ → More tools → WebAuthn and tick “Enable virtual authenticator environment” (set it to internal / resident-key + UV supported). Register first, then authenticate.
